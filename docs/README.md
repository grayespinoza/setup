# Shell Setup Scripts

## Usage

### Arch Linux

```shell
curl -fsSLO https://raw.githubusercontent.com/grayespinoza/setup/main/arch.sh
cat arch.sh
bash arch.sh
```

### Dotfiles

```shell
curl -fsSLO https://raw.githubusercontent.com/grayespinoza/setup/main/dotfiles.sh
cat dotfiles.sh
bash dotfiles.sh
```

### Hyprland

```shell
curl -fsSLO https://raw.githubusercontent.com/grayespinoza/setup/main/hyprland.sh
cat hyprland.sh
bash hyprland.sh
```
